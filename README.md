# Platfooorm Movie - Deep Links

GitHub Pages-hosted landing pages and Android App Links verification for the
Platfooorm Movie app.

- `docs/film.html` — dynamic movie landing page (TMDB-powered)
- `docs/index.html` — homepage
- `docs/.well-known/assetlinks.json` — Android App Links verification

## URL format

```
https://developer262391-lang.github.io/Platfooorm-Movie/film.html?id={tmdbId}
https://developer262391-lang.github.io/Platfooorm-Movie/film.html?id={tmdbId}&type=tv
https://developer262391-lang.github.io/Platfooorm-Movie/film.html?id={tmdbId}&ref=tiktok
```
